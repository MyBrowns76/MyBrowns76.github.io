FizzBuzzQix

In this project we are given one test, then we had to write the rest on our own. We took the old Fizz Buzz kata out for a ride, but we then added a new feature to develop: Qix. The functionality was that I had to test out for FizzBuzzQix by: Given an integer, return a String representation of that integer. Given an integer that is a multiple of three, return a String that contains the word "Fizz". Given an integer that is a multiple of five, return a String that contains the word "Buzz". Given an integer that is a multiple of seven, return a String that contains the word "Qix". If multiple words were used, add them in order of Fizz, Buzz, then Qix.
